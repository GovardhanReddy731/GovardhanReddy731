## 🚀 Featured Projects

### 🚧 Accident Detection & Alerting System
**`Jun 2025 – Aug 2025`**

Real-time computer vision system that detects road accidents from live and recorded video streams and automatically alerts emergency contacts.

**Key Features**
- Real-time **vehicle detection** using YOLOv8
- **Multi-object tracking** across frames with DeepSORT
- **Human posture analysis** via MediaPipe Pose Estimation
- **Temporal action recognition** (LSTM) to understand behavior over time
- **Decision fusion module** that combines model outputs to reduce false positives
- FastAPI backend supporting video upload, offline processing, and live stream analysis
- **Twilio SMS integration** for instant alerts with timestamp and location

**Engineering Challenges**
- Fusing outputs from multiple independent models (detection, tracking, pose, action) into a single reliable decision
- Balancing real-time performance with detection accuracy on live video streams
- Minimizing false positives in a safety-critical alerting pipeline

**Tech Stack:** `Python` `FastAPI` `OpenCV` `YOLOv8` `DeepSORT` `MediaPipe` `LSTM` `WebSockets` `Twilio` `HTML/CSS/JS`

### 🩺 Skin Allergy Detection Using Deep Learning
**`Jun 2024 – Aug 2024`**

Medical image classification system that detects skin allergies from user-uploaded images and provides preliminary diagnosis support.

**Key Features**
- **EfficientNet-B4 transfer learning** for feature extraction — **91.7% classification accuracy**
- Image preprocessing pipeline: resizing, normalization, data augmentation
- RESTful APIs for prediction requests and report storage
- Dashboard showing predicted class, confidence score, and recommended precautions
- Secure authentication and cloud deployment for real-time use

**Engineering Challenges**
- Achieving high accuracy on a limited, imbalanced medical imaging dataset
- Designing a preprocessing pipeline robust to varied lighting/image quality from user uploads
- Presenting model confidence responsibly in a health-adjacent application

**Tech Stack:** `Python` `TensorFlow` `OpenCV` `Flask` `React` `Tailwind CSS` `MongoDB` `Cloud Deployment`
